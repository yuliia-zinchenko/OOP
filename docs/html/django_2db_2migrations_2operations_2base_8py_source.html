15class Operation:
16 """
17 Base class for migration operations.
18
19 It's responsible for both mutating the in-memory model state
20 (see db/migrations/state.py) to represent what it performs, as well
21 as actually performing it against a live database.
22
23 Note that some operations won't modify memory state at all (e.g. data
24 copying operations), and some will need their modifications to be
25 optionally specified by the user (e.g. custom Python code snippets)
26
27 Due to the way this class deals with deconstruction, it should be
28 considered immutable.
29 """
