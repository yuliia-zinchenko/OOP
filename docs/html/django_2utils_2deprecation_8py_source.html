42 Handles the deprecation paths when renaming a method.
43
44 It does the following:
45 1) Define the new method if missing and complain about it.
46 2) Define the old method if missing.
47 3) Complain whenever an old method is called.
48
49 See #15363 for more details.
50 """

62 new_method_name = renamed_method[1]
